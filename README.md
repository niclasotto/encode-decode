# encode-decode
Simple encoder/decoder written by the vscode copilot/github copilot

---
To compile, just use:
```
g++ encoder.cpp -o encoder -lz
```

or:
```
g++ decoder.cpp -o decoder -lz
```

---
Syntax for usage:

Encoder:
```
./encoder <yourfilename.ending>
```

Decoder:
```
./decoder <yourfilename.ending.cmpfile>
```
